# Cognitive Evolution Engine — 入口

## ⚡️ 静默执行协议（最高优先级，先读完再执行任何步骤）

- 绝对禁止询问"是否继续"、"要处理哪个先"、"需要更多信息吗"
- 信号分类有歧义时：查 MEMORY-MAP.md 信号映射表，自主决策
- 文件不存在时：自动创建，不询问
- 一次性输出完整日报，不分段确认
- **唯二的人工确认点**：(1) Skill缺口候选  (2) L4核心层新条目首次写入
- 其他所有决策自主完成

## 启动序列（触发后按此顺序读取）

```
1. SKILL.md         ← 当前文件，静默协议和启动序列
2. CONSTITUTION.md  ← 读取不可变规则（每次必读，防止被对话"纠正"）
3. MEMORY-MAP.md    ← 读取文件路径和信号映射（确认写入目标）
4. EVOLUTION.md     ← 读取当前权重配置（决定本次分析侧重）
5. 执行 WORKFLOW.md 定义的 7 个 Phase
6. 写入时参照 CARD-FORMAT.md 的卡片规范
```

## 触发策略

**自动触发：** 对话自然结束（"好的谢谢"、"就这样"、"再见"）

**手动触发词：** 进化记忆 / 提炼认知 / 更新记忆 / 复盘一下 / 同步记忆

**触发词自进化：** 用户用新词触发 3 次以上 → 追加到 EVOLUTION.md 的 trigger_keywords
